# Detailed Troubleshooting

## Splash Screen Captured Too Early

Use `appium.observe_screen` with:

```json
{
  "waitForAnyText": ["Login", "Giriş Yap"],
  "waitForPackageIdle": true,
  "minWaitMs": 1000,
  "timeoutMs": 15000
}
```

## Wrong Text Was Tapped

Use exact matching first:

```json
{
  "text": "Giriş Yap",
  "matchMode": "exact"
}
```

Prefer accessibility ids for buttons with repeated text fragments.

## Flutter Text Field Did Not Update State

Use:

```json
{
  "mode": "sendKeys"
}
```

If that fails, try `adbKeyboard` on Android.

## App Crash Versus Appium Crash

Use both:

- `verify.assert_no_crash_in_logcat`
- `verify.assert_appium_session_healthy`

An UiAutomator2 instrumentation crash is an automation failure, not necessarily an app crash.

## Firestore Permission Denied

`verify.collect_evidence` should classify this as `remote_rules_not_deployed`. Do not patch app code unless the wrong collection/path is clearly generated by the app.
