# MobiLoop MCP v0.1.0-alpha.7

Alpha CI maintenance release.

## Highlights

- Updated GitHub Actions major versions:
  - `actions/checkout@v6`
  - `actions/setup-node@v6`
  - `actions/upload-artifact@v7`
- Resolved the GitHub Actions Node.js 20 runtime deprecation warning emitted by `actions/upload-artifact@v4`.
- Verified the action update with GitHub-hosted CI and CodeQL.

## Validation

The update was accepted after:

- PR checks passed for CI and CodeQL.
- Main CI passed after merge.
- Main CodeQL passed after merge.

The Android Fixture E2E workflow remains queued because no self-hosted Android runner is currently available. That is tracked as an operations issue rather than a blocker for this CI maintenance release.
