# MobiLoop MCP v0.1.0-alpha.8

Alpha iOS orchestration release.

## Highlights

- Added `orchestrator.run_ios_validation_loop`.
- Added iOS simulator app discovery from `ios.build_app` DerivedData output.
- Added Flutter Android/iOS quickstart documentation.
- Added `examples/flutter-ios-validation-loop.json`.
- Added mock Appium integration coverage for the iOS orchestrator.
